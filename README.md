# Gadget Mart

Gadget Mart is a responsive e-commerce website built with React.js and Tailwind CSS.

## Features

* Browse products by category
* Product details page
* Add products to cart
* Update product quantity
* Remove products from cart
* Checkout page
* Responsive design for mobile, tablet, and desktop
* Local Storage support for cart data

## Technologies Used

* React.js
* React Router
* Tailwind CSS
* JavaScript (ES6)
* Local Storage

## Live Demo
https://gadget-mart-ecommerce-react-with-ta.vercel.app/

## Installation
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
